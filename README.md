What I Did
Selected a sample phishing email for analysis.
Examined the email content for phishing indicators.
Identified a generic greeting, urgency, and impersonation of a trusted bank.
Checked for requests to click a login link and noted that such requests are suspicious.
Observed that the email attempted to create trust by using security-related language.
Documented all phishing indicators and summarized the findings in a report.

Key Phishing Indicators Found
Generic greeting ("Dear People's Bank Client")
Impersonation of a trusted bank
Request to sign in using an email link
Urgent/security-related message
Potential risk of credential theft
Lack of personalized account information
